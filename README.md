# Google-Clone

### React based simple search engine powered by Google api.

### Stack
- Tailwind css
- React v18
- React Router DOM v6
- Rapid API

### To Run locally
- npm install
- Place .env file on root directory, containning REACT_APP_RAPIDAPI_KEY
- npm start
